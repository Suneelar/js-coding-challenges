# JavaScript Coding Challenges

A collection of JavaScript coding challenges and mini tasks created to strengthen core programming concepts and improve problem-solving skills.  
This repository is ideal for beginners and intermediate learners who want to practice and enhance their JavaScript skills.

- Beginner to intermediate level JS tasks  
- Focus on core concepts like variables, loops, functions, arrays, and objects  
- DOM manipulation and event handling exercises  
- Clean, structured, and easy-to-understand code  
- Real-world and interview-style problems

Topics Covered
- Variables & Data Types  
- Functions & Loops  
- Arrays & Objects  
- DOM Manipulation  
- Event Handling  
- Conditional Statements  
- ES6 Features
